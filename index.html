P1: Zelda
Below you can see the group effort that Ryan Duan and I put to create a replica of the classic Legend of Zelda game for the NES!

MY CONTRIBUTIONS:

  • Handled room transitions: Moving from room to room, disabling enemy movement in between rooms, creating locked doors, etc.
  • Created the Boomerang weapon: Coded the behavorial script, handled its interactions with other objects, dealt with enemy stun, and more.
  • Created the Sprites for the new mechanic: I love drawing! Drew new sprites, such as a portal gun, orange/blue portals, and new projectiles.
  • Created most collectables: Implemented mob drops like hearts, rupees, keys, bombs, and special ones like heart containers and the boomerang.
  • Implemented most of the sound effects: Different pick-up sounds, attacking with a weapon, bomb explosion, enemy death, and more.
  • To just name a few :D!
TECHNOLOGY USED:
  • Coding Language: C# as done in Unity
  • Game Engine: Unity v.2022.3.43
  • Drawing Program to make sprites: Krita
  • Code Editor: Visual Studio Community 2019

P2: Ice Cube Crawl
Here is my creation for project 2! I was inspired by games like N-Step Steve and Can of Wormholes to make a movement based puzzle game.
I wanted the mechanics to be simple to grasp without need for instructions yet complex when they all work together.

MY CONTRIBUTIONS:

  • Hand drew all the sprites + art used in the game using Krita.
  • Designed and implemented all the code using the EventBus architecture and other techniques.
  • Searched for fitting audio to use for all the soundeffects off of freesound.org (I did not create any sounds)
  • Designed all the levels used in the game with player growth in mind, so that players would learn mechanics on their own.
TECHNOLOGY USED:
  • Coding Language: C# as done in Unity
  • Game Engine: Unity v.2022.3.43
  • Drawing Program to make sprites: Krita
  • Code Editor: Visual Studio Community 2019

P3: Rainy Day Rush
Run, jump, and glide your way through this 2d side scrolling platformer with nothing but your trusty umbrella as you try to make it to your interview. But be careful, you won't get that job easily; there are dangers lurking around every corner. Brave the weather and keep your suit clean to look your best for the interview - Hope you get the job!
