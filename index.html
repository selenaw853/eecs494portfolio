P1: Zelda
Below you can see the group effort that Ryan Duan and I put to create a replica of the classic Legend of Zelda game for the NES!

MY CONTRIBUTIONS:

  • Handled room transitions: Moving from room to room, disabling enemy movement in between rooms, creating locked doors, etc.
  • Created the Boomerang weapon: Coded the behavorial script, handled its interactions with other objects, dealt with enemy stun, and more.
  • Created the Sprites for the new mechanic: I love drawing! Drew new sprites, such as a portal gun, orange/blue portals, and new projectiles.
  • Created most collectables: Implemented mob drops like hearts, rupees, keys, bombs, and special ones like heart containers and the boomerang.
  • Implemented most of the sound effects: Different pick-up sounds, attacking with a weapon, bomb explosion, enemy death, and more.
  • To just name a few :D!
TECHNOLOGY USED:
  • Coding Language: C# as done in Unity
  • Game Engine: Unity v.2022.3.43
  • Drawing Program to make sprites: Krita
  • Code Editor: Visual Studio Community 2019

P2: Ice Cube Crawl
